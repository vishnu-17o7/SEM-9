🔬 How It Works

28 features are extracted from each URL, capturing structural patterns that distinguish phishing from legitimate websites:

Length-based
  • URL length, path length, domain length
  • Digit ratio, special character ratio
  • URL entropy (randomness measure)
Structural
  • Number of dots, hyphens, slashes, @ symbols
  • Number of subdomains, path depth
  • Presence of IP address, port, HTTPS
Content-based
  • Suspicious keywords (login, verify, secure...)
  • URL shortener detection
  • Double-slash redirect patterns
Domain-based
  • TLD length, has-www flag
  • Domain-based features
