# Exercise 01 - Coimbatore Georeferencing

Open `project/coimbatore_georef.qgz` for the final georeferenced view. `project/work-georef.qgz` preserves the intermediate comparison workflow.

- `data/source` contains the scanned map, original raster, control-point inputs, and Coimbatore MBTiles.
- `data/processed` contains the modified and georeferenced raster stages.
- `outputs` contains the exported PDF and PNG results.

The projects use relative paths, so the complete exercise can be moved as a folder.
