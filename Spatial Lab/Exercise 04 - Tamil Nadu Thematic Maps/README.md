# Exercise 04 - Tamil Nadu Thematic Maps

Open `project/exercise-04-tamil-nadu-thematic-maps.qgz`. The processed GeoPackage and source CSVs support four presentation maps:

1. disease incidence per 100,000 using graduated colors;
2. disease composition using pie charts;
3. disease counts using bar charts; and
4. total cases using proportional symbols.

Final PDF and PNG exports are in `outputs/final`. The earlier thematic-map project, practice GeoPackage, and exports are retained in `archive/legacy-version`.

The disease values are synthetic laboratory practice data, not real public-health statistics.
